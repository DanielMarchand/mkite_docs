====================
Basic mkite tutorial
====================

This tutorial complements the :doc:`15-min quickstart <quickstart>` and exemplifies how to use mkite for:

1. Executing recipes
2. Running jobs with the mkwind daemon
3. Organizing the database metadat a with projects and experiments
4. Importing files into the production database
5. Creating, submitting, and parsing jobs
6. Creating workflows
