=================
Advanced tutorial
=================

Once you have mastered how to use mkite as a user, you may be interested in using mkite as a developer.
This tutorial will cover the following topics:

- Creating new recipes for simulations
- Creating new engines
- Creating new scheduler interfaces
- Performing advanced queries to retrieve data from the production database
- Understanding serialization and deserialization
- Changing the database schema
- Taking advantage of the flexible database schema
